# MotoBarna Website Goal

Build the V1 MotoBarna website as a polished, modern Astro static landing page for a Barcelona MotoGymkhana and motorcycle community. The site should convert riders into the Telegram group while clearly balancing technical MotoGymkhana, Sunday parking-lot hangs, and weekend motorcycle trips around Barcelona and Catalunya.

The shared understanding is captured in [facts.md](facts.md). The implementation path is captured in [plan.md](plan.md).

Done means the accepted facts are implemented or explicitly verified, the site builds successfully, responsive browser QA passes on mobile and desktop, Telegram CTAs point to `https://t.me/motobarna`, Mixpanel tracks Telegram CTA clicks, SEO foundations are present, and the page includes English and Russian language routes.
